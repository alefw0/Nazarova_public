# Implementación del algoritmo de Nazarova-Roiter

Mi idea con este proyecto es la de profundizar en el apasionante mundo del álgebra, pero buscando intersecciones con la computación. 
En este primer acercamiento a mi gran objetivo, no uso objetos interesantes de la teoría de la computación (como los lenguajes o los autómatas), simplemente implemento un algoritmo en su forma matricial. Con esto, busco adquirir expericencia para eventualmente tratar con temas un poco más ambiciosos.

Este proyecto se compone de dos partes:
- Carpera docs: Acá está todo el contenido de documentación, que decidí hacerlo lo más simple y plano posible, para escalarlo fácilmente a otros tipos de documentos.
- Carpeta src: Acá estará el algoritmo de Nazarova-Roiter, escrito en pyhton.
